# open-admin





## 环境变量

|          变量名          |                             描述                             |
| :----------------------: | :----------------------------------------------------------: |
|   **OPEN_ADMIN_ROOT**    |           项目运行根目录,系统自动设置,用户无需配置           |
|    **OPEN_ADMIN_ID**     |  服务唯一标识,常用于分布式环境。当用户不指定时,则会随机生成  |
| **OPEN_ADMIN_LOG_LEVEL** | 控制台日志输出级别,优先级：命令行参数>环境变量>默认配置.[debug, info,warn,error] |
|                          |                                                              |
|                          |                                                              |
